# NovaTech Onboarding Guide

## Getting Started (First 30 Minutes)

1. Accept your email invitation and set a password
2. Complete the security questionnaire (required for compliance)
3. Set up two-factor authentication (mandatory for all users)
4. Join your assigned workspace

## Setting Up Your First Connection

1. Navigate to Connections > New Connection
2. Select your data source type from the catalog
3. Enter connection credentials (hostname, port, username, password or OAuth)
4. Click "Test Connection" to verify connectivity
5. Choose sync frequency: real-time, hourly, or daily
6. Select tables or collections to sync
7. Click "Activate" to start the initial sync

## Understanding Workspaces

Workspaces are isolated environments for teams. Each workspace has its own connections, queries, and access controls. Users can belong to multiple workspaces but data does not cross workspace boundaries.

## User Roles

- Viewer: Read-only access to dashboards and query results
- Analyst: Can create and run queries, build dashboards
- Admin: Full access including connection management and user administration
- Owner: Workspace owner with billing and deletion permissions
