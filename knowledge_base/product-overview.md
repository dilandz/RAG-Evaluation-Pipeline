# NovaTech Product Overview

## What is NovaTech Platform?

NovaTech Platform is an enterprise data integration solution that connects disparate data sources into a unified analytics layer. It supports real-time streaming ingestion from Kafka, batch processing from S3 and GCS, and direct database connections to PostgreSQL, MySQL, and MongoDB.

## Key Features

- Real-time data streaming with sub-second latency
- Automatic schema detection and evolution handling
- Built-in data quality checks with configurable thresholds
- Role-based access control (RBAC) with SSO integration
- RESTful API for programmatic access to all platform features

## Supported Integrations

NovaTech connects to over 50 data sources out of the box including cloud warehouses (Snowflake, BigQuery, Redshift), databases (PostgreSQL, MySQL, MongoDB, DynamoDB), streaming platforms (Kafka, Kinesis, Pub/Sub), and file storage (S3, GCS, Azure Blob).

## Pricing Tiers

- Starter: Up to 10 data sources, 1M events/day, 3 users
- Professional: Up to 50 data sources, 10M events/day, 25 users
- Enterprise: Unlimited sources, unlimited events, unlimited users, dedicated support
